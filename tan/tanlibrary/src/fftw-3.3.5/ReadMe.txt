For Windows:
download and unzip FFTW binaries from:
ftp://ftp.fftw.org/pub/fftw/fftw-3.3.5-dll64.zip
to folder fftw-3.3.5-dll64.

For Linux and Mac:
download and build shared float libraries from sources for FFTW from: 
ftp://ftp.fftw.org/pub/fftw/fftw-3.3.5.tar.gz
Hint:  use ./configure --enable-shared --enable-float
